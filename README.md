
# Noura Foods - Data Analytics & Machine Learning Project

این پروژه پیاده‌سازی مسیر کامل پردازش، تحلیل و پیش‌بینی داده (End-to-End) برای مجموعه فرضی Noura Foods در حوزه FMCG است. داده‌های این پروژه به صورت سینتتیک و با اعمال دقیق قوانین واقعی کسب‌وکار (Business Rules) طراحی شده‌اند.

---

## فازهای اجرایی

### فاز ۱: پردازش و مدل‌سازی داده (ETL)

* **مدیریت مقادیر پرت (Outliers):** سقف‌گذاری داده‌های نامتعارف تراکنش‌ها.
* **پاک‌سازی:** بازسازی و جایگزینی مقادیر از دست رفته (Missing Values) با تجمیع داده‌ها.
* **مدل‌سازی ابعادی:** طراحی دیتابیس تحلیلی بر اساس Star Schema (جداول Fact و Dimension).
* **بارگذاری:** انتقال داده‌های ساختاریافته به دیتابیس MySQL از طریق SQLAlchemy.

### فاز ۲: تحلیل تجاری (SQL)

* **شاخص‌های کلان:** محاسبه مجموع سفارشات، درآمد کل و ۵ محصول پرفروش.
* **روندها:** محاسبه رشد ماهانه (MoM) با استفاده از Window Functions و مقایسه ارزش سفارشات (AOV).
* **رفتار مشتری:** بررسی توزیع سنی خریداران، محاسبه نرخ وفاداری و شناسایی مشتریان VIP.

### فاز ۳: تحلیل آماری (Inferential Statistics)

* **A/B Testing:** مقایسه علمی رفتار کاربران در خریدهای ارگانیک نسبت به کمپین‌ها.
* **Correlation Analysis:** اثبات عدم تأثیر معنادار «سن مشتری» بر «ارزش سبد خرید» با آزمون Spearman جهت بهینه‌سازی بودجه تارگتینگ تبلیغات.

### فاز ۴: پیش‌بینی ریزش مشتریان (Machine Learning)

* **Feature Engineering:** استخراج متغیرهای رفتاری مانند تازگی (Recency) و فرکانس خرید از روی تاریخ تراکنش‌ها.
* **مدل‌سازی:** آموزش مدل Logistic Regression برای پیش‌بینی Churn.
* **استراتژی بیزینسی (Cost-Sensitive Learning):** تنظیم مدل روی Recall معادل ۸۴٪ (با پذیرش Accuracy کل ۷۳٪) جهت حداکثر کردن شکار مشتریان در حال ریزش؛ با این منطق که هزینه از دست دادن مشتری به مراتب بالاتر از هشدارهای کاذب (False Positives) است.
