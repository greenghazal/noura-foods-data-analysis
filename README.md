# Noura Foods - Data Analytics & Engineering Project

این پروژه شامل پیاده‌سازی مسیر پردازش و تحلیل داده برای مجموعه Noura Foods (فعال در حوزه FMCG) است. تمرکز این مستند بر پاک‌سازی ساختاریافته‌ی داده‌ها، مدل‌سازی ابعادی و استخراج گزارش‌های تجاری با استفاده از پایگاه داده رابطه‌ای است.

## ابزارها و تکنولوژی‌ها
* **زبان برنامه‌نویسی:** Python (Pandas, Numpy)
* **پایگاه داده:** MySQL 
* **مفاهیم توسعه:** ETL, Medallion Architecture, Star Schema, SQLAlchemy

---

## فاز اول: پردازش و مدل‌سازی داده (ETL) - وضعیت: تکمیل شده
در این فاز، داده‌های خام پردازش شده و برای تحلیل در انبار داده آماده شدند. مراحل طی شده عبارتند از:
1. **مدیریت مقادیر پرت (Outliers):** شناسایی و سقف‌گذاری مقادیر نامتعارف بر اساس الگوی مصرف مشتریان نهایی (B2C).
2. **بازسازی داده‌های ناقص:** محاسبه و جایگزینی مقادیر از دست رفته (Missing Values) در فاکتورها از طریق تجمیع مجدد جداول ریزتراکنش‌ها.
3. **مدل‌سازی:** طراحی پایگاه داده تحلیلی بر مبنای Star Schema و تفکیک داده‌ها به جداول Fact و Dimension.
4. **انتقال داده:** برقراری اتصال به سرور MySQL از طریق موتور SQLAlchemy و بارگذاری جداول نهایی در پایگاه داده.

*(اسکریپت‌های پایتون مربوط به این بخش در فایل نوت‌بوک موجود است).*

## فاز دوم: تحلیل تجاری با SQL - وضعیت: تکمیل شده
در این فاز، پرسش‌های تجاری به کوئری‌های SQL تبدیل شده و خروجی‌ها به همراه نتایج تحلیلی (Insights) مستند شدند.

### سطح مبتدی (شاخص‌های کلان)
* **نمای کلی کسب‌وکار:** استخراج مجموع سفارشات، اقلام فروخته شده و کل درآمد شرکت.
* **شناسایی محصولات کلیدی (Top Sellers):** رتبه‌بندی ۵ محصول پرفروش بر اساس حجم فروش برای مدیریت موجودی انبار.

### سطح متوسط (روندها و عملکردها)
* **اثربخشی کانال‌های بازاریابی:** مقایسه درآمد حاصل از فروش ارگانیک در برابر کمپین‌های تبلیغاتی مختلف.
* **روند درآمد ماهانه (MoM Growth):** محاسبه نوسانات و اختلاف درآمد هر ماه نسبت به ماه قبل با استفاده از توابع تحلیلی (Window Functions).
* **تحلیل فصلی و رفتار هفتگی:** بررسی اوج تقاضا در ماه‌های مختلف سال و شناسایی پرترافیک‌ترین روزهای هفته.
* **میانگین ارزش سفارش (AOV):** بررسی ارزش سبد خرید مشتریان بر اساس کمپین‌های جذب.

### سطح پیشرفته (تحلیل رفتار مشتری)
* **توزیع سنی مشتریان:** دسته‌بندی خریداران به گروه‌های سنی (جوان، بزرگسال و...) با استفاده از `CASE WHEN` و محاسبه سهم درآمدی هر گروه.
* **نرخ بازگشت و وفاداری (Retention):** طبقه‌بندی خریداران به یک‌باره، دو-باره و مشتریان وفادار (۳+ خرید) از طریق جداول موقت (`WITH`).
* **شناسایی مشتریان VIP:** استخراج ۱۰ مشتری برتر با بیشترین دفعات تکرار خرید و بالاترین ارزش مالی برای برنامه‌های وفاداری.
